# NeverStop - Premium Photo Platform

A full-stack Unsplash-style photo platform built with:

- React + Tailwind
- Node.js + Express
- Unsplash API
- JWT Authentication
- Razorpay Integration
- Smart Explore Algorithm

## Features

- Smart Explore Feed
- Infinite Scroll
- Image Preloading
- Blur Modal Animation
- Fullscreen Viewer
- Login & Signup
- Favorites
- Professional UI

## Setup

### Backend

cd backend
npm install
npm start

### Frontend

cd frontend
npm install
npm run dev
